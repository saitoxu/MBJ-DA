## Usage

### Requirements

- [pyenv](https://github.com/pyenv/pyenv)
- [Poetry](https://github.com/python-poetry/poetry)
- You need to install python (>=3.8 and <3.11) via pyenv in advance.

### Setup

```sh
$ poetry env use 3.8.10 # please specify your python version
$ poetry install
```
